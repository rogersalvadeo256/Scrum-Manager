# Screenshots

Esta pasta armazena as capturas de tela do Scrum Manager usadas na documentação.

## Como gerar as capturas

1. Suba o stack completo (PostgreSQL, Redis, RabbitMQ, backend, frontend).
2. Acesse `http://localhost:5173` e faça login.
3. Tire as capturas das seguintes telas e salve nesta pasta com os nomes abaixo:

| Arquivo | Descrição |
|---|---|
| `dashboard-overview.png` | Aba "Visão geral" do dashboard principal |
| `scrum-dashboard.png` | Seção "Dashboard do Scrum Master" dentro de um projeto |
| `project-board.png` | Board Kanban de tarefas de um projeto |
| `scalar-docs.png` | Página `http://localhost:8080/docs` com Scalar |
